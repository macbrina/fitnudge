## 3. Rarity-Based Icon Styles

Achievement badges have rarity levels. Consider these styles when searching:

| Rarity | Suggested Style |
|--------|-----------------|
| **common** | Simple flat icon, single color, minimal |
| **rare** | Slightly detailed, maybe silver/metallic |
| **epic** | Gold accent, more ornate, premium feel |
| **legendary** | Sparkle, glow, crown, diamond accents |

**Search terms for rarity:** badge rare, badge epic, legendary badge, gold badge, silver badge, premium badge

---

## 4. General Tips

- Prefer **SVG** format for scalability in React Native
- Use **flat** or **line** style for consistency with the app
- For badges: search "badge icon", "achievement badge", "medal icon"
- Add "minimal" or "simple" to narrow results
- Flaticon: filter by "Free" if needed; check license for commercial use
- Freepik: same – verify attribution/commercial license
